Class RapidContext.Widget.TableColumn

.Widget.TableColumn(attrs) → {Widget}

The table column widget class. Used to provide a sortable data table column, using a <th> HTML element for the header (and rendering data to <td> HTML elements).

Creates a new data table column widget.

Parameters:
Name Type Description
attrs {Object} the widget and node attributes
Properties
Name Type Attributes Description
title {string} the column title
field {string} the data property name
type {string} <optional>
 the data property type, one of "string", "number", "date", "time", "datetime", "boolean" or "object" (defaults to "string")
sort {string} <optional>
 the sort direction, one of "asc", "desc", "none" (disabled) or null (unsorted)
maxLength {number} <optional>
 the maximum data length, overflow will be displayed as a tooltip
key {boolean} <optional>
 the unique key value flag, only to be set for a single column per table
tooltip {string} <optional>
 the tooltip text to display on the column header
cellStyle {string} <optional>
 the CSS styles or class names to set on the rendered cells
renderer {function} <optional>
 the function that renders the converted data value into a table cell, called as renderer(<td>, value, data) with the DOM node, field value and data object as arguments
Returns:
{Widget} the widget DOM node
Examples:

JavaScript

let attrs1 = { field: "id", title: "Identifier", key: true, type: "number" };
let attrs2 = { field: "name", title: "Name", maxLength: 50, sort: "asc" };
let attrs3 = { field: "modified", title: "Last Modified", type: "datetime" };
let col1 = RapidContext.Widget.TableColumn(attrs1);
let col2 = RapidContext.Widget.TableColumn(attrs2);
let col3 = RapidContext.Widget.TableColumn(attrs3);
let exampleTable = RapidContext.Widget.Table({}, col1, col2, col3);

User Interface XML

<Table id="exampleTable" w="50%" h="100%">
  <TableColumn field="id" title="Identifier" key="true" type="number" />
  <TableColumn field="name" title="Name" maxLength="50" sort="asc" />
  <TableColumn field="modified" title="Last Modified" type="datetime" />
</Table>

addAll(…child)

Adds one or more children to this widget. This method will flatten any arrays among the arguments and ignores any null or undefined arguments. Any DOM nodes or widgets will be added to the end, and other objects will be converted to a text node first. Subclasses should normally override the addChildNode() method instead of this one, since that is the basis for DOM node insertion.

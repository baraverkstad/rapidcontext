<static> RapidContext.SVG.moveToTop(node)
Moves a node to the top of the SVG drawing order (i.e. z-index). Note that this will only have effect on other SVG DOM nodes with the same parent node. Otherwise, the node must be moved down in the SVG DOM tree by changing parent node.
Parameters:
{Node/String} node
the SVG DOM node or unique id
<static> RapidContext.SVG.moveToBottom(node)
Moves a node to the bottom of the SVG drawing order (i.e. z-index). Note that this will only have effect on other SVG DOM nodes with the same parent node. Otherwise, the node must be moved up in the SVG DOM tree by changing parent node.
Parameters:
{Node/String} node
the SVG DOM node or unique id

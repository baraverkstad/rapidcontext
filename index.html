What Is RapidContext?

RapidContext is a platform for building and hosting web applications. It makes it easy to build, deploy and manage your intranet tools. Or to build new ones.

  • See the documentation introduction for more info.

Documentation

All the documentation below is also available in the download package.

  • Reference Manual – Reference documentation with examples and usage information.
  • JavaScript API – Client-Side JavaScript API for building apps.
  • Java API (Plug-In API) – Server-Side Java API for building connectivity plug-ins.
